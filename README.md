# Gatekeeper
A redshift permissions management tool to help you automate and version control data access

## Getting Started
````text
pip install gatekeeper
````

##### Initialize the project
```text
gatekeeper init
```

##### Accessing help
```text 
gatekeeper <command> -h
```
